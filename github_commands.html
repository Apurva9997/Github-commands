    GIT

  1. git --version -- Check Git version
  2. Git Configuration -- git config --global user.name "Apurva Garg"
  3. git config --global user.email "apoorvgarg01@yahoo.com"
  4. git config --list -> Displays current user credentials
  5. GIT help -> git verb --help or git help verb
  6. GIT initialization to an existing project -> git init
  7. ls -la : List all the files in the current project folder
  8. git status : shows the status of the project
  9. git reset : to remove files from the staging area
  10. git commit : to commit all the changes to the local repository
  11. git commit -m "messageGoesHere" : to commit all the changes to the local repository with a message
  12. git log : Shows the commits and author and messages(commit)
  13. touch gitignore : creates a git ignore file where we can use .fileextension or wildcards(e.g *.pyc)
  14. Remove git from an project -> (In gitbash) rm -rf .git: This will remove furthur version control from our project
  15. git add -A : Add all the untracked file to the staging area
  16. git reset : Removes everything from the staging area
